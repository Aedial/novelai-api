class LowLevel[source]¶

Bases: object

Low level API for NovelAI. This class is not meant to be used directly, but rather through NovelAIAPI.low_level.

The most relevant methods are:

  • login

  • generate

  • generate_image

  • get_keystore

  • download_objects
